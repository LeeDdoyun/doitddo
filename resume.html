저는 삶이란 경험의 집약체라고 생각합니다. 그렇기에 경험의 깊이와 다양성을 중요하게 여깁니다. 하지만 사람이 직접 경험할 수 있는 것들에는 한계가 있기에,
사람은 타인의 경험을 기반으로 부족한 경험을 보완하곤 합니다.

일상에서 우리는 다양한 서비스를 마주합니다. 하지만 모든 서비스가 나에게 딱 맞는 것은 아닙니다. “어떻게 하면 사용자에게 더 좋은 경험을 전달할 수 있을까?”
라는 고민은, 동일한 서비스를 직접 경험하지 않은 사용자에게 제 경험을 기반으로 부족함을 채워가는 과정이라고 생각합니다.

졸업 프로젝트로 앱 서비스 기획을 할 때였습니다. 앱 서비스 기획을 위해 UI UX 사용자 경험 공부를 토대로 고객이 서비스를 이용하면서 느끼는 불편을 고민하는
시간을 가졌었습니다. 이러한 과정은 초기 모델의 보완점을 자체적으로 피드백할 수 있는 시간이었으며, 이것을 토대로 UI UX 콘셉트를 도출하여 서비스 기획에
더 체계적으로 다가갈 수 있었습니다. 이후 디자인 형태를 직접 모색하고 설계하여 좋은 결과를 만들어냈습니다. 이 과정을 통해 컬러, 아이콘, 폰트 작은 요소 하나
까지 사용자 입장과 브랜드 콘셉트에 어울리는지 고민하며 선정하고 배치하는 것의 중요성에 대해서 깨닫게 되었습니다. 사용자의 필요를 데이터 기반으로 파악하고,
니즈를 우선으로 심미적 요소를 디자인해 만족도를 높일 수 있었던 것입니다.
